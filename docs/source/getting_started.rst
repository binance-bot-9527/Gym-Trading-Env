入门指南
=======

安装
----

Gym Trading Env 支持 Windows、Mac 和 Linux 上的 Python 3.9+。你可以使用 pip 安装：

.. code-block:: console

   pip install gym-trading-env

或者使用 git 克隆：

.. code-block:: console
   
   git clone https://github.com/ClementPerroud/Gym-Trading-Env


导入
----

你可以这样导入 Gym Trading Env：

.. code-block:: python

   import gym_trading_env
